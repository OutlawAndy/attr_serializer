# AttrSerializer
easy JSON serialization for ActiveRecord models

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'attr_serializer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install attr_serializer
